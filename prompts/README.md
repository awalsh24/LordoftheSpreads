# Prompts

System prompts and tool specs for the LLM agent layer.

This folder fills in once the strategy code exists and we know what the agent needs to see and do.

Planned files:
- `agent_system.md` — main system prompt for the orchestrator agent
- `tools.md` — tool specs the agent can call (read_book, set_spread, halt, etc.)
- `regime_classifier.md` — prompt for the "what kind of market is this right now" sub-call
